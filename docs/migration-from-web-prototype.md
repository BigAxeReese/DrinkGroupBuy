# Migration From Web Prototype

## Status

This document is archived.

The old Web prototype has already been deleted after user confirmation:

- `frontend/`
- `server.js`
- `src/`
- `data/`

The project no longer keeps a Web reference prototype in the repository.

## Current Direction

Development now continues from:

- `mobile/` for the Android-first app
- `database/` for the development schema
- future `backend/` for API implementation
- `docs/current-progress.md` for handoff context

## Historical Note

Earlier planning used the Web prototype as a reference for screen flow and data requirements. That reference value has now been replaced by the mobile prototype and documentation.

Do not restore deleted Web files unless the user explicitly requests it.
